# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-04-01

### Added

- Initial public release of Forge for managing local coding-agent workflows on macOS
- Project and workspace management with lightweight APFS copy-on-write clones
- Agent-aware terminal sessions with Ghostty-backed rendering, tabs, and split panes
- Built-in Git status, diff, and review workflows
- Bundled `forge` CLI for IPC and automation
- Makefile-based development workflow and GitHub release scaffolding
